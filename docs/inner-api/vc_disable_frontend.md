# vc_disable_frontend()

To disable frontend editor on your site call `vc_disable_frontend()` function from your functions.php file.

Available SINCE 4.0

```php
<?php vc_disable_frontend(); ?>
```

## Parameters

None.

## Example
How to disable frontend editing mode in WPBakery Page Builder.

```php
<?php
vc_disable_frontend();
?>
```
